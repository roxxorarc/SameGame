## Getting Started



### Prerequisites

* [Maven](https://maven.apache.org/install.html)
  

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/roxxorarc/SameGame
   ```
2. Enter the root folder
   ```sh
   cd SameGame
   ```
3. Compile and execute
   ```
   mvn clean javafx:run
   ```
